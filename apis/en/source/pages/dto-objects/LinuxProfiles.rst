linuxProfiles
=============

Provides a list of :ref:`linuxprofile-object` objects.

Attributes
~~~~~~~~~~

The list of attributes for ``linuxProfiles`` are:

	* ``linuxProfiles``: the list of :ref:`linuxprofile-object` objects
	* ``total`` (long): the total number of objects
