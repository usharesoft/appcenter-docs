Appliance Partition Table Disk Partition API Resources
======================================================

	* :ref:`partitionTableDiskPartition-getAll`: Retrieves all the physical partitions in a disk
	* :ref:`partitionTableDiskPartition-create`: Creates a new physical partition in a disk
	* :ref:`partitionTableDiskPartition-deleteAll`: Removes all the physical partitions from a disk
	* :ref:`partitionTableDiskPartition-get`: Retrieves the information of a physical partition in a disk
	* :ref:`partitionTableDiskPartition-delete`: Removes a physical partition from a disk
	* :ref:`partitionTableDiskPartition-update`: Updates a physical partition in a disk
	* :ref:`partitionTableDiskLogicalPartition-getAll`: Retrieves all the logical partitions in a physical partition
	* :ref:`partitionTableDiskLogicalPartition-create`: Create a logical partition in a physical partition
	* :ref:`partitionTableDiskLogicalPartition-deleteAll`: Removes all the logical partitions from a physical partition
	* :ref:`partitionTableDiskLogicalPartition-get`: Retrieves the information of a logical partition
	* :ref:`partitionTableDiskLogicalPartition-delete`: Removes a logical partition from a physical partition
	* :ref:`partitionTableDiskLogicalPartition-update`: Updates a logical partition information in a physical partition
