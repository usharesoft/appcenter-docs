targetFormats
=============

Provides a list of :ref:`targetformat-object` objects.

Attributes
~~~~~~~~~~

The list of attributes for ``targetFormats`` are:

	* ``targetFormats``: the list of :ref:`targetformat-object` objects
	* ``total`` (long): the total number of objects
