Appliance Configuration API Resources
=====================================

	* :ref:`applianceBootScript-getAll`: Retrieves all the boot scripts for a particular appliance
	* :ref:`applianceBootScript-create`: Adds a new boot script object to an appliance
	* :ref:`applianceBootScript-deleteAll`: Removes all of the boot scripts attached to an appliance
	* :ref:`applianceBootScript-delete`: Removes a boot script attached to an appliance
	* :ref:`applianceBootScript-get`: Retrieves a boot script that is attached to an appliance
	* :ref:`applianceBootScript-update`: Updates a boot script that is attached to an appliance
	* :ref:`applianceBootScript-download`: Downloads a boot script file
	* :ref:`applianceBootScript-downloadFile`: Downloads a boot script file
	* :ref:`applianceBootScript-upload`: Uploads a boot script file
