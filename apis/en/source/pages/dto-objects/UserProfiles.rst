userProfiles
============

Provides a list of :ref:`userprofile-object` objects.

Attributes
~~~~~~~~~~

The list of attributes for ``userProfiles`` are:

	* ``userProfiles``: the list of :ref:`userprofile-object` objects
	* ``total`` (long): the total number of objects
