	* :ref:`osTemplate-getAll`: Retrieves all the templates available for an operating system
	* :ref:`osTemplate-update`: Updates the meta-data information of an OS template
	* :ref:`osTemplate-delete`: Removes an OS template from an operating system
	* :ref:`osTemplate-get`: Retrieves the meta-data of an OS template
	* :ref:`osTemplatePkgs-get`: Retrieves the list of native packages for an OS template
