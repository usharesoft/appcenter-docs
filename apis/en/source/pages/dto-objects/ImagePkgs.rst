imagePkgs
=========

Provides a list of :ref:`imagepkg-object` objects.

Attributes
~~~~~~~~~~

The list of attributes for ``imagePkgs`` are:

	* ``imagePkgs``: the list of :ref:`imagepkg-object` objects
	* ``total`` (long): the total number of objects
