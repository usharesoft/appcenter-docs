Appliance API Resources
=======================

	* :ref:`applianceLogo-delete`: Deletes the logo binary from the specified appliance
	* :ref:`applianceLogo-download`: Downloads the logo binary for the appliance
	* :ref:`applianceLogo-downloadFile`: Downloads the logo binary for the appliance
	* :ref:`applianceLogo-upload`: Uploads a logo for the specified appliance
	* :ref:`appliance-clone`: Creates a new appliance by cloning all the meta-data of an existing appliance
	* :ref:`appliance-create`: Creates a new appliance in a user's ``Appliance Library``
	* :ref:`appliance-delete`: Deletes the specified appliance
	* :ref:`appliance-get`: Gets the information for an appliance
	* :ref:`appliance-getAll`: Gets all the appliances created by a particular user
	* :ref:`appliance-update`: Updates the meta-data information of an appliance
