galleryTemplates
================

Provides a list of :ref:`gallerytemplate-object` objects.

~~~~~~~~~~

The list of attributes for ``galleryTemplates`` are:

	* ``galleryTemplates``: the list of :ref:`gallerytemplate-object` objects
