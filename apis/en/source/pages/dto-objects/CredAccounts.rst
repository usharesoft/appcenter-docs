credAccounts
============

Provides a list of :ref:`credaccount-object` objects.

Attributes
~~~~~~~~~~

The list of attributes for ``credAccounts`` are:

	* ``credAccounts``: the list of :ref:`credaccount-object` objects
	* ``total`` (long): the total number of objects
