scannedFiles
============

Provides a list of :ref:`scannedfile-object` objects.

Attributes
~~~~~~~~~~

The list of attributes for ``scannedFiles`` are:

	* ``scannedFiles``: the list of :ref:`scannedfile-object` objects
	* ``total`` (long): the total number of objects
