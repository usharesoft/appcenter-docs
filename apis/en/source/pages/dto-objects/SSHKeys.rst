.. _sshkeys-object:

sshKeys
=======

Provides a list of :ref:`sshkey-object` objects.

Attributes
~~~~~~~~~~

The list of attributes for ``sshKeys`` are:

	* ``sshKeys``: the list of :ref:`sshkey-object` objects
	* ``total`` (long): the total number of objects
