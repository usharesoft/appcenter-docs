bootScripts
===========

Provides a list of :ref:`bootscript-object` objects.

Attributes
~~~~~~~~~~

The list of attributes for ``bootScripts`` are:

	* ``bootScripts``: the list of :ref:`bootscript-object` objects
	* ``total`` (long): the total number of objects
