Artifact Accounts API Resources
===============================

	* :ref:`orgArtifactCredAccount-create`: Creates a new artifact account that can be used by all the users in the organization
	* :ref:`orgArtifactCredAccount-getAll`: Retrieves all organization artifact accounts
	* :ref:`orgArtifactCredAccount-get`: Retrieve information of an artifact account
	* :ref:`orgArtifactCredAccount-update`: Update the meta-data of an artifact account
	* :ref:`orgArtifactCredAccount-delete`: Remove an artifact account from an organization
	* :ref:`userArtifactCredAccount-create`: Creates a new artifact account for an user
	* :ref:`userArtifactCredAccount-getAll`: Retrieve all artifact accounts for a user
	* :ref:`userArtifactCredAccount-get`: Retrieve information of an artifact account
	* :ref:`userArtifactCredAccount-update`: Update the meta-data of an artifact account
	* :ref:`userArtifactCredAccount-delete`: Remove an artifact account from an user
