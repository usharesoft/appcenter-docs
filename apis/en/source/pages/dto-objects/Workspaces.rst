.. _workspaces-object:

workspaces
==========

Provides a list of :ref:`workspace-object` objects.

Attributes
~~~~~~~~~~

The list of attributes for ``workspaces`` are:

	* ``workspaces``: the list of :ref:`workspace-object` objects
	* ``total`` (long): the total number of objects
