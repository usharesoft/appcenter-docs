Appliance OS Profile API Resources
==================================

	* :ref:`applianceOSProfileApplications-get`: Retrieves an application contained in the OS profile
	* :ref:`applianceOSProfileApplications-getAll`: Retrieves all the applications contained in the OS profile
	* :ref:`applianceOSProfilePkg-getAll`: Retrieves all the native packages contained in the OS profile
	* :ref:`applianceOSProfilePkg-updateAll`: Updates the native packages contained in an OS profile
	* :ref:`applianceOSProfileServices-get`: Retrieves a service contained in the OS profile
	* :ref:`applianceOSProfileServices-getAll`: Retrieves all the services contained in the OS profile
	* :ref:`applianceOSProfile-create`: Adds a new OS profile to an appliance
	* :ref:`applianceOSProfile-delete`: Removes an OS profile from an appliance
	* :ref:`applianceOSProfile-get`: Retrieves an OS profile from an appliance
	* :ref:`applianceOSUpdates-get`: Retrieves a list of possible package updates for all the packages contained in an OS profile
	* :ref:`applianceOSUpdates-save`: Updates the OS profile package list with any available updates
