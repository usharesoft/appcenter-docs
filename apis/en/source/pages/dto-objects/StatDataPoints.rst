statDataPoints
==============

Provides a list of :ref:`statdatapoint-object` objects.

Attributes
~~~~~~~~~~

The list of attributes for ``statDataPoints`` are:

	* ``statDataPoints``: the list of :ref:`statdatapoint-object` objects
	* ``total`` (long): the total number of objects
