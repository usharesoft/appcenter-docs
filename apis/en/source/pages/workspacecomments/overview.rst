Workspace Comments API Resources
================================

	* :ref:`workspaceComment-create`: Posts a comment to a workspace
	* :ref:`workspaceComment-delete`: Removes a comment from a workspace
	* :ref:`workspaceComment-deleteAll`: Removes all the comments for a workspace
	* :ref:`workspaceComment-dislike`: Dislike a comment in a workspace
	* :ref:`workspaceComment-get`: Retrieve a comment from a workspace
	* :ref:`workspaceComment-like`: Like a comment in a workspace
	* :ref:`workspaceComment-reply`: Create a reply to a comment in a workspace
	* :ref:`workspaceComment-reportAbuse`: Report a comment is abusive
	* :ref:`workspaceComment-update`: Updates the information of a comment
	* :ref:`workspaceComments-getAll`: Retrieves all the comments posted in a workspace
