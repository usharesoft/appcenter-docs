btrfsSubvolumes
===============

Provides a list of :ref:`btrfssubvolume-object` objects.

Attributes
~~~~~~~~~~

The list of attributes for ``btrfsSubvolumes`` are:

	* ``btrfsSubvolumes``: the list of :ref:`btrfssubvolume-object` objects
	* ``total`` (long): the total number of objects
