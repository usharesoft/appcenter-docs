Repositories API Resources
==========================

	* :ref:`repositoryOS-getAll`: Retrieve all distributions registered to a repository
	* :ref:`repositoryUpdateStatus-get`: Retrieves the status of a repository update
	* :ref:`repositoryUpdate-create`: Triggers a new package update process for a specific repository, within an organization
	* :ref:`repositoryUpdate-getAll`: Returns all the updates for a repository
	* :ref:`repository-create`: Creates a new repository in an organization
	* :ref:`repository-delete`: Remove a repository from an organization
	* :ref:`repository-getAll`: Retrieves all the repositories for an organization
	* :ref:`repository-update`: Update the repository in an organisation
