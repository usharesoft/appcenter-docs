Appliance Partition Table API Resources
=======================================

	* :ref:`partitionTableBtrfsStoragePool-create`: Creates a new btrfs storage pool in a partitioning table
	* :ref:`partitionTableBtrfsStoragePool-delete`: Removes a btrfs storage pool from a partitioning table
	* :ref:`partitionTableBtrfsStoragePool-deleteAll`: Removes all the btrfs storage pools from a partitioning table
	* :ref:`partitionTableBtrfsStoragePool-get`: Retrieves the information of a btrfs storage pool
	* :ref:`partitionTableBtrfsStoragePool-getAll`: Retrieves all the btrfs storage pools for a partitioning table
	* :ref:`partitionTableBtrfsStoragePool-update`: Updates the information of a btrfs storage pool
	* :ref:`partitionTableBtrfsSubvolume-create`: Creates a new btrfs subvolume in a partitioning table
	* :ref:`partitionTableBtrfsSubvolume-delete`: Removes a btrfs subvolume from a partitioning table
	* :ref:`partitionTableBtrfsSubvolume-deleteAll`: Removes all the btrfs subvolumes from a partitioning table
	* :ref:`partitionTableBtrfsSubvolume-get`: Retrieves the information of a btrfs subvolume
	* :ref:`partitionTableBtrfsSubvolume-getAll`: Retrieves all the btrfs subvolumes for a partitioning table
	* :ref:`partitionTableBtrfsSubvolume-update`: Updates the information of a btrfs subvolume
