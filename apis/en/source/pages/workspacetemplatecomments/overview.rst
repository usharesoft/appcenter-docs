Workspace Template Comments API Resources
=========================================

	* :ref:`workspaceTemplateComment-create`: Posts a comment to a shared appliance template in a workspace
	* :ref:`workspaceTemplateComment-getAll`: Retrieves all the comments posted in a shared appliance template
	* :ref:`workspaceTemplateComment-deleteAll`: Removes all the comments for a shared appliance template
	* :ref:`workspaceTemplateComment-reply`: Create a reply to a comment
	* :ref:`workspaceTemplateComment-get`: Retrieve a comment from a workspace
	* :ref:`workspaceTemplateComment-update`: Updates the information of a comment
	* :ref:`workspaceTemplateComment-delete`: Removes a comment from a shared appliance template
	* :ref:`workspaceTemplateComment-reportAbuse`: Report a comment is abusive
	* :ref:`workspaceTemplateComment-like`: Like a comment in a workspace
	* :ref:`workspaceTemplateComment-dislike`: Dislike a comment in a workspace
