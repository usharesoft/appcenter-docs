Workspace Members API Resources
===============================

	* :ref:`workspaceMember-invite`: Invites one or more users to become members of a workspace
	* :ref:`workspaceMember-getAll`: Retrieves all the members of a workspace
	* :ref:`workspaceMember-deleteList`: Removes one or more users as members of a workspace
	* :ref:`workspaceMember-updateList`: Updates the membership roles of one or more users in a workspace
	* :ref:`workspaceMember-delete`: Removes a user as a member of a workspace
	* :ref:`workspaceMember-update`: Updates the membership role of an user in a workspace
