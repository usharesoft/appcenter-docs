applianceImports
================

Provides a list of :ref:`applianceImport-object` objects.

Attributes
~~~~~~~~~~

The list of attributes for ``applianceImports`` are:

	* ``applianceImports``: the list of :ref:`applianceImport-object` objects
	* ``total`` (long): the total number of objects
