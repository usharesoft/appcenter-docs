Appliance Install Users and Groups API Resources
================================================

	* :ref:`installProfileGroup-create`: Adds a new OS group in an install profile
	* :ref:`installProfileGroup-delete`: Removes an OS group from an install profile
	* :ref:`installProfileGroup-deleteAll`: Removes all the OS groups from an install profile
	* :ref:`installProfileGroup-get`: Retrieves the meta-data information of an OS group registered in an install profile
	* :ref:`installProfileGroup-getAll`: Retrieves all the OS groups registered in an install profile
	* :ref:`installProfileGroup-update`: Updates an OS group registered in an install profile
	* :ref:`installProfileUser-create`: Adds a new OS user in an install profile
	* :ref:`installProfileUser-delete`: Removes an OS user from an install profile
	* :ref:`installProfileUser-deleteAll`: Removes all the OS users from an install profile
	* :ref:`installProfileUser-get`: Retrieves the meta-data information of an OS user registered in an install profile
	* :ref:`installProfileUser-getAll`: Retrieves all the OS users registered in an install profile
	* :ref:`installProfileUser-update`: Updates an OS user registered in an install profile
