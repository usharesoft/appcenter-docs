scannedPackages
===============

Provides a list of :ref:`scannedpackage-object` objects.

Attributes
~~~~~~~~~~

The list of attributes for ``scannedPackages`` are:

	* ``scannedPackages``: the list of :ref:`scannedpackage-object` objects
	* ``total`` (long): the total number of objects
